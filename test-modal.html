Teste do Modal de Cadastro

Para testar o modal de erro:

  1. Acesse: http://localhost:3002/
  2. Clique na aba "Cadastrar"
  3. Teste os seguintes cenários:

Cenário 1 - Campos vazios:

Deixe algum campo vazio e clique em "Cadastrar"

Resultado esperado: Modal de erro com mensagem "Por favor, preencha todos os campos."

Cenário 2 - Senhas não coincidem:

Preencha todos os campos mas coloque senhas diferentes

Resultado esperado: Modal de erro com mensagem "As senhas não coincidem."

Cenário 3 - Senha curta:

Use uma senha com menos de 6 caracteres

Resultado esperado: Modal de erro com mensagem "A senha deve ter pelo menos 6 caracteres."

Cenário 4 - RG inválido:

Use um RG que não existe na base, por exemplo: 99999

  • RG: 99999
  • Nome: Teste
  • Posto: 2º Ten
  • Quadro: QOMBM
  • Unidade: 1º GSFMA
  • Senha: 123456
  • Confirmar Senha: 123456

Resultado esperado: Modal de erro com botão WhatsApp para contato

Cenário 5 - RG já cadastrado:

Use o RG: 12961 (que já está cadastrado como admin)

Resultado esperado: Modal de erro com mensagem "Este RG já está cadastrado na plataforma."
